# Contributing to Spotify Sorter

Thank you for your interest in contributing to Spotify Sorter! I welcome contributions from everyone.

## Reporting Bugs

- **Search Existing Issues**: Before creating a new issue, please search the [open issues](https://github.com/JamyPatel/Spotify-Sorter/issues) to see if someone else has already reported it.
- **Use the Bug Report Template**: When creating a new issue, use the "Bug Report" template which guides you through providing the necessary information.
- **Provide Details**: details are key! Include your OS, app version, and steps to reproduce.

## Suggesting Features

- I use the "Feature Request" template for new ideas.
- explain *why* this feature would be useful and how it should work.

## Development Setup

1.  **Prerequisites**:
    - Node.js (v18+)
    - Rust & Cargo
    - [Tauri Prerequisites](https://tauri.app/v2/guides/prerequisites/) for Windows.

2.  **Clone & Install**:
    ```bash
    git clone https://github.com/JamyPatel/Spotify-Sorter.git
    cd spotify-sorter
    npm install
    ```

3.  **Run Locally**:
    ```bash
    npm run tauri dev
    ```

## Pull Requests

1.  Fork the repository.
2.  Create a new branch (`git checkout -b feature/amazing-feature`).
3.  Commit your changes.
4.  Push to the branch.
5.  Open a Pull Request.

Please make sure your code follows the existing style and conventions.
